## Usage

```tsx
import ArrayField from "@/components/array-field";
```

```tsx
<ArrayField
  name="emails"
  label="Email Addresses"
  description="Add up to 5 email addresses where we can contact you."
>
  {(name, { remove }) => (
    <TextField
      key={`${name}.address`}
      name={`${name}.address`}
      placeholder="Enter email"
      addonRight={
        <Button type="button" variant="ghost" onClick={remove}>
          <X />
        </Button>
      }
    />
  )}
</ArrayField>
```
